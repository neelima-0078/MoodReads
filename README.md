# MoodReads
MoodReads is a simple website that helps you find books based on your mood. Just pick how you’re feeling—cozy, curious, sad, or motivated—and the site suggests books that match that vibe. Each recommendation includes a short summary and a link to start reading online, making it easy and fun to discover your next book.

## Features
- Mood-based book recommendations
- Clean modern UI
- Links to public-domain books

## Built With
- HTML
- CSS
- JavaScript
